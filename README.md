# nexalive-website

NexaLive — Enterprise Communication Platform Website.

Built with [Astro](https://astro.build) + React + TailwindCSS.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 🏗️ Build & Deploy

```bash
npm run build
```

Deployed on Cloudflare Pages → [nexalive.ai](https://nexalive.ai)
